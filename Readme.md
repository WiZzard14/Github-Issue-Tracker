Q: What is the difference between var, let, and const?

A: Those are are use to declare variables in JS but using var can be redeclared and update, using let value can be updated but can't be redeclared and using const value can't be updated or can't be redeclared.

Q: What is the spread operator (...)?

A: It's used to expand elements of an array or object.

Q: What is the difference between map(), filter(), and forEach()?

A: Those all are used to work with arrays but using map basically create a new array and change every element and using filter it create new array but using selected elements, foreach() run a function for each element and doesn't return a new array.
